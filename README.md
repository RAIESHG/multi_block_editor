# DXF Template Editor

A Streamlit web application that allows users to:
1. Upload DXF template files
2. View and select blocks from the template
3. Edit block attributes in a table format
4. Generate multiple DXF files with different block names

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
streamlit run app.py
```

## Features

- DXF file upload and processing
- Block selection interface
- Attribute editing in table format
- Multiple file generation support
- Streamlit cloud deployment ready
